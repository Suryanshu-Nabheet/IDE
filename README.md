**CodeX** is a next-generation, premium AI-powered IDE designed for elite developers. Featuring the sophisticated **Oxocarbon** theme and an offline-first architecture, CodeX delivers a professional, distraction-free environment for high-performance engineering.

---

## ✨ Features

### 🧠 Intelligence First

-   **Native AI Assistance** - Seamlessly integrated AI for code generation, debugging, and complex refactoring.
-   **Deep Language Intelligence** - Powered by advanced Language Server Protocol (LSP) for precise completions and navigation.
-   **Context-Aware Chat** - Chat with your codebase directly within the editor.

### 🎨 Elite Aesthetics

-   **Oxocarbon Prime** - A bespoke, premium IBM Carbon-inspired monochromatic theme with subtle amber accents.
-   **Micro-Animation Engine** - Every interaction is smooth, intentional, and visually rewarding.
-   **Uncluttered Interface** - Designed for maximum real estate, focusing entirely on your code.

### 🛠️ Professional Tooling

-   **Multimodal Terminal** - High-performance, integrated multi-tab terminal with full shell support.
-   **Unified Sidebars** - Elegant search, file tree, and AI panels that respect your visual focus.
-   **Enterprise-Grade Stability** - Built on a hardened Electron architecture for reliable daily use.

### 🔒 Privacy & Performance

-   **Offline-First** - Your core workflow resides on your machine. No mandatory cloud dependencies.
-   **Ultra-Fast Performance** - Optimized file indexing and rendering for projects of any size.

## 🚀 Quick Start

### Prerequisites

-   **Node.js** v18+
-   **npm** v9+
-   **Git**

### Installation

```bash
# 1. Clone the elite repository
git clone https://github.com/Suryanshu-Nabheet/CodeX.git
cd CodeX

# 2. Execute professional setup
./setup.sh  # macOS/Linux
.\setup.ps1  # Windows

# 3. Launch the experience
npm start
```

## 🏗️ Technical Architecture

CodeX is built on a modern, high-performance stack:

-   **Core Orchestrator**: [Electron](https://www.electronjs.org/)
-   **Visual Interface**: [React 18](https://reactjs.org/) & [Tailwind CSS](https://tailwindcss.com/)
-   **State Integrity**: [Redux Toolkit](https://redux-toolkit.js.org/)
-   **Editor Engine**: [CodeMirror 6](https://codemirror.net/)
-   **Type Intelligence**: [TypeScript](https://www.typescriptlang.org/)

## 🌍 Language Support
